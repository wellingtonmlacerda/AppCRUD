# App_Teste
App em .net core 7.0. 

## Como inicar o app:
Acesse o projeto de banco e execulte os scripts dentro de DBAppCRUD/Scripts.

Depois aponte a conexão do arquivo appsettings.json para o banco que você criou.

Pronto, agora é só iniciar a aplicação.
